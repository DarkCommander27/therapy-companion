# Data Retention Template

This is a template to help facilities define retention periods aligned to local policy and regulations.

## Recommended principle
- Retain the **minimum necessary** data for the minimum necessary time.

## Suggested buckets
- Conversations (raw): ___ days/months
- Briefings (summaries): ___ months/years
- Incident logs: ___ months/years
- Audit logs: ___ months/years

## Deletion and exports
- Define who can export data (recommended: limited roles only).
- Define deletion policy and approval workflow.
